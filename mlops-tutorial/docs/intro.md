# Welcome to Zero-to-Hero MLOps

Master Kubernetes, infrastructure automation, and machine learning operations in 9 progressive phases.

## What You'll Learn

This comprehensive tutorial takes you from zero to expert in ML/LLMOps:

- **Phase 1**: Kubernetes Foundations (6-8 hours)
- **Phase 2**: Infrastructure as Code (4-6 hours)
- **Phase 3**: Chaos Engineering & Diagnostics (4-5 hours)
- **Phase 4**: Autonomous Self-Healing (5-6 hours)
- **Phase 5**: Model Training Pipeline (8-10 hours)
- **Phase 6**: GPU Scheduling & Cost Optimization (6-8 hours)
- **Phase 7**: AutoML & Hyperparameter Tuning (6-8 hours)
- **Phase 8**: Distributed Training - Simulation (4-6 hours)
- **Phase 9**: Distributed Training - Production (6-8 hours)

**Total**: 40-60 hours spread over 4-8 weeks

## Target Audience

- Intermediate developers (Python, Docker basics, heard of Kubernetes)

## Prerequisites

Before starting, make sure you have:

- **Docker Desktop** or **Rancher Desktop** installed and running
- **Terminal/CLI** comfort (basic commands: cd, ls, cat)
- **Python 3.10+** installed
- **4GB RAM** and **20GB disk space**
- **2-3 hours** uninterrupted time per session (or split into smaller chunks)
